## Practical 03 Portfolio in React

---

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jenith-Panchal-Simform/Practical-03-Portfolio.git
   cd Practical-03-Portfolio
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development Server**
   ```bash
   npm run dev
   ```
